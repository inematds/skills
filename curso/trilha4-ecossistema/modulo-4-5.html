Simples, token unico no header.

Delegated auth, tokens temporarios.

üì• Webhooks Recebidos

Webhooks sao callbacks HTTP que sistemas externos enviam quando eventos ocorrem. Permitem integracao em tempo real sem polling.

Anatomia de um Webhook

  • ‚Ä¢URL: Endpoint que recebe o POST
  • ‚Ä¢Payload: JSON com dados do evento
  • ‚Ä¢Signature: Hash para verificar autenticidade
  • ‚Ä¢Retry: Re-envio em caso de falha

Verificando Signature

import hmac
import hashlib

def verify_signature(payload, signature, secret):
    expected = hmac.new(
        secret.encode(),
        payload.encode(),
        hashlib.sha256
    ).hexdigest()
    return hmac.compare_digest(expected, signature)

‚ö†Ô∏è Tratamento de Erros

Integracoes robustas precisam tratar erros graciosamente. APIs podem falhar por diversos motivos: rede, rate limits, erros de servidor.

Status Codes

  • 2xx - Sucesso
  • 4xx - Erro do cliente
  • 5xx - Erro do servidor
  • 429 - Rate limited

Retry Strategy

  • ‚Ä¢ Exponential backoff
  • ‚Ä¢ Max retries (3-5)
  • ‚Ä¢ Jitter para evitar thundering herd
  • ‚Ä¢ Circuit breaker para falhas persistentes

üîß MCP Servers

Model Context Protocol (MCP) e o protocolo da Anthropic para conectar Claude a ferramentas externas de forma padronizada.

Componentes MCP

  • ‚Ä¢Tools: Funcoes que o agente pode chamar
  • ‚Ä¢Resources: Dados que o agente pode ler
  • ‚Ä¢Prompts: Templates pre-definidos

MCP Server Exemplo

from mcp.server import Server

server = Server("weather-service")

@server.tool()
async def get_weather(city: str) -> str:
    """Retorna clima atual de uma cidade."""
    # Implementacao
    return f"Clima em {city}: 25C, ensolarado"

# GitHub PR Reviewer

Para revisar um PR:

1. Buscar diff: `gh pr diff $PR_NUMBER`
2. Analisar mudancas
3. Postar comentarios: `gh pr review $PR_NUMBER --comment -b "..."`
