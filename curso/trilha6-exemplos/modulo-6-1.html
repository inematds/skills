üîÆ anthropics/skills - O Repositorio da Anthropic

O repositorio anthropics/skills e a fonte oficial de skills de referencia para Claude Code. Mantido pela propria Anthropic, define os padroes e melhores praticas que toda a comunidade segue. Aqui voce encontra skills exemplares que demonstram como estruturar instrucoes, definir permissoes e criar fluxos eficientes.

O formato oficial utiliza o frontmatter YAML com campos como name, description e allowed-tools, seguidos pelo corpo da skill em Markdown. Esse padrao garante que o Claude Code interprete corretamente as instrucoes e aplique as permissoes de ferramentas adequadas.

# Code Review Expert

Ao revisar codigo, siga este checklist:
1. **Corretude** - A logica esta correta?
2. **Seguranca** - Existem vulnerabilidades?
3. **Performance** - Pode ser otimizado?
4. **Legibilidade** - O codigo e claro?
5. **Testes** - Ha cobertura adequada?

ü§ñ openai/skills - Catalogo do OpenAI Codex

A OpenAI disponibiliza instrucoes e skills otimizadas para seus agentes, especialmente o Codex. O formato utiliza AGENTS.md como arquivo principal de instrucoes, similar ao CLAUDE.md da Anthropic. Estudar essas skills revela como a OpenAI estrutura instrucoes para maximizar a performance de seus modelos.

O ecossistema OpenAI foca em function calling e system prompts como mecanismo principal de customizacao. Skills nesse contexto definem como o agente deve usar tools, estruturar respostas e lidar com ambiguidades. Os padroes sao transferiveis para qualquer plataforma.

‚ñ≤ vercel-labs/agent-skills - Vercel e Next.js

O vercel-labs/agent-skills e o repositorio da Vercel com skills focadas em desenvolvimento frontend e full-stack moderno. Skills para Next.js, Vercel AI SDK, React Server Components e deploy na plataforma Vercel formam o nucleo dessa colecao.

Essas skills sao particularmente valiosas por capturarem padroes que mudam frequentemente no ecossistema React/Next.js. Em vez de memorizar APIs que mudam a cada versao, as skills mantem o agente atualizado com as melhores praticas vigentes.

Exemplo: Skill Next.js App Router

---
name: nextjs-app-router
description: Expert em Next.js App Router. Use para criar
  paginas, layouts, server components e API routes.
allowed-tools: Read, Write, Edit, Bash, Glob
---

# Next.js App Router Expert

## Regras Fundamentais
- Sempre use Server Components por padrao
- Use 'use client' apenas quando necessario (eventos, state, browser APIs)
- Prefira fetch() em Server Components sobre useEffect
- Use loading.tsx e error.tsx para UX
- Metadata deve ser exportada de page.tsx e layout.tsx

O repositorio supabase/agent-skills contem skills para trabalhar com a plataforma Supabase: PostgreSQL, Row Level Security, Edge Functions, Realtime e Auth. Essas skills transformam o agente em um especialista em backend-as-a-service.
